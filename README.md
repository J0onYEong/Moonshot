# Moonshot
학습용 SwiftUI 토이프로젝트입니다.

# 사용된 기술
- Resizing Image
- GeometryReader
- ScrollView
- LazyStack
- NavigationLink
- 계층형 데이터(JSON) 디코딩 방식및 에러처리 
- Date formatter
- extention을 활용하여 Color인스턴스 추가
- Dark mode
